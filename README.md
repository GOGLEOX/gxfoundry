# GOGLEOX Foundry

> Forged in fire, crafted for utility.

Lightweight Minecraft mods built for Forge and NeoForge.

GOGLEOX Foundry is an independent Minecraft mod workshop focused on practical gameplay improvements, survival utility, multiplayer usability, and long-term maintainability. Projects are intentionally designed around controlled scope, lightweight architecture, and compatibility-conscious development.

## Philosophy

GOGLEOX Foundry prioritizes:

- Utility-focused gameplay
- Lightweight implementation
- Server-friendly behavior
- Modpack compatibility
- Maintainable long-term support
- Controlled feature scope

Projects avoid unnecessary bloat, oversized systems, excessive GUI complexity, and maintenance-heavy mechanics whenever possible.

## Current Focus

Current development focuses primarily on:

- Inventory utility
- Survival realism
- Multiplayer quality-of-life
- Lightweight automation
- Practical server utilities

Forge 1.20.1 currently serves as the primary development baseline due to ecosystem maturity and broad modpack adoption.

## Data-Driven Architecture

The website uses a lightweight static architecture:

- HTML for structure
- CSS for branding and presentation
- JavaScript for rendering and interactivity
- JSON for content management

This allows the catalog, roadmap, and version outlook systems to remain maintainable without requiring backend infrastructure or framework dependencies.
